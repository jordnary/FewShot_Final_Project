# Few-Shot Learning Reproduction Project

本项目面向“小样本学习”方向，目标是在 PyTorch/LibFewShot 框架下完成 2023-2026 年顶会论文复现，并提交代码、实验报告 PDF 与展示材料。

当前推荐主线：

1. 先跑通 LibFewShot 环境、数据集和经典基线。
2. 选择 1-2 篇 LibFewShot 尚未覆盖的近年顶会方法进行复现。
3. 在统一设置下比较原论文结果、复现结果和消融实验。
4. 整理为可贡献的 classifier/backbone/config 结构，尽量贴近 LibFewShot 贡献规范。

目录说明：

- `00_docs/`: 项目计划、论文笔记和阅读记录。
- `01_environment/`: 环境配置、依赖和服务器命令。
- `02_libfewshot/`: LibFewShot 安装、运行和代码贡献记录。
- `03_datasets/`: 数据集下载、划分和路径说明。
- `04_baseline_experiments/`: 基线实验日志。
- `05_paper_reproduction/`: 论文复现代码、配置和结果。
- `06_improvement/`: 改进实验和消融分析。
- `07_report/`: 最终报告与展示材料。
